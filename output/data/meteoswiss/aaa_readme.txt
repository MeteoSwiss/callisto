This folder contains the data downloaded automatically using meteoswiss_realtime_data2csv.py, which is
run through the System Scheduler every 15 minutes. Every csv file, organized by day, contains the 
following (from the stations at Locarno-Monti)
   - temperature [degC] 2 meters above ground, 
   - the precipitation [mm], and 
   - the global radiation (e.g. solar irradiance) [W/m2].
